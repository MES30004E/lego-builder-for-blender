# LEGO Builder for Blender Help Centre

LEGO Builder for Blender is an early Blender 5.1+ extension for building a
native LEGO modelling environment inside Blender.

The current version provides LEGO Library setup, LDraw library validation, and a
metadata-only part index. It does not import LEGO geometry, create meshes, show a
part browser, generate thumbnails, snap parts, export models, or download
libraries yet.

## Where to Start

1. Install the extension in Blender.
2. Create or select a LEGO Library folder.
3. Point the LDraw library path at a valid LDraw library.
4. Refresh the library index.

Use the navigation to find setup steps, troubleshooting notes, and short
answers to common questions.
