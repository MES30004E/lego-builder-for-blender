# Installation

LEGO Builder for Blender is currently installed from a local extension ZIP.

## Requirements

- Blender 5.1 or newer.
- A LEGO Builder for Blender extension ZIP.

## Install from Disk

1. Open Blender.
2. Go to `Edit > Preferences > Extensions`.
3. Choose `Install from Disk`.
4. Select the LEGO Builder extension ZIP.
5. Enable the extension if Blender does not enable it automatically.

After installation, open the 3D View sidebar and look for the `LEGO` tab.

## Updating During Early Development

During early development, install newer ZIP builds manually. If Blender removes
add-on preferences during uninstall or reinstall, re-select your LEGO Library
and LDraw library paths.
