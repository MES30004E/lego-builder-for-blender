# Getting Started

This guide describes the current early setup flow.

## First Use

1. Install LEGO Builder for Blender from a local extension ZIP.
2. Open Blender and enable the extension if Blender asks you to.
3. Open the `LEGO` sidebar tab in the 3D View.
4. Use the settings button to open the add-on preferences.
5. Choose a LEGO Library folder.
6. Create the LEGO Library folders.
7. Configure an LDraw library path.
8. Refresh the library index.

## Current Capabilities

- Shows a compact LEGO Builder sidebar.
- Stores user settings in Blender add-on preferences.
- Creates a managed LEGO Library folder structure.
- Validates LDraw library folders that contain `parts/` and `p/`.
- Builds an in-memory metadata index from `.dat` files.

## Current Limits

LEGO Builder does not yet import geometry, place parts, search parts, generate
thumbnails, download LDraw libraries, create materials, snap parts, or export
models.
