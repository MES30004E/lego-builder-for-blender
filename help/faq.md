# FAQ

## Can LEGO Builder Import Parts Yet?

No. The current extension indexes part metadata only.

## Does Refresh Library Download LDraw?

No. Downloads and update management are future work.

## Does LEGO Builder Create Thumbnails?

No. Thumbnail generation is not implemented yet.

## Where Are Settings Stored?

Selected paths are stored in Blender add-on preferences. The LEGO Library folder
also contains a marker file so it can be recognized again later.

## Is This a BrickLink Studio Replacement?

Not yet. The long-term goal is a Blender-native LEGO building environment, but
the current project is still in foundation milestones.
