# Future: Brick Builder

Brick Builder is a possible future name for the dedicated LEGO modelling
environment inside Blender.

The current extension does not create a custom Blender workspace or dedicated
building layout. The sidebar remains the compact control surface for status and
quick actions.

Future work may include:

- Dedicated building layout.
- Part placement tools.
- Snapping and connection feedback.
- Object inspection.
- Color and material controls.

These features are not implemented yet.
